# AI-Powered Research Paper Assistant

An intelligent RAG-based application that helps students, researchers, and professionals understand research papers through natural language interaction.

## Features

- PDF Upload
- Semantic Search
- Research Paper Q&A
- AI Summarization
- Citation Generation
- Multi-Paper Comparison
- Quiz Generation

## Tech Stack

- Python
- Streamlit
- LangChain
- FAISS
- Sentence Transformers
- Gemini API

## Architecture

PDF → Chunking → Embeddings → FAISS → Retriever → Gemini → Answer
